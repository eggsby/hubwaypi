HubwAyPI

The hubway is the bike share system in Boston, MA. It is comprised of a series of stations with bike docks around the city and surrounding towns that users can use for 30 minutes periods.

This API is an open source system to make hubway information accessible and easy to play with. The goal of this project is to make it easier for developers to play with and make things with the information hidden away behind the hubway bike system

All calls are made from the url http://thpcod.es/hubwaypi

API Calls
  • All Stations
  • Closest Station[s]
  • Single Station
  • + Station Details
  • Number of bikes

To contribute to this API/see the code behind it, please checkout: https://github.com/thethp/hubwaypi
